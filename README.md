# HuijiAppSimulator
A chrome extension to browse huiji wiki in 'app mode'

# Install
1. Download [crx file](https://github.com/lianzhao/HuijiAppSimulator/raw/master/dist/HuijiAppSimulator.crx)
2. Open (chrome://extensions/)
3. Drag and drop the crx file into that page

# Get Start
1. Open a huiji wiki page in 'normal' mode, e.g. (http://coppermind.huiji.wiki/wiki/%E9%A3%93%E5%85%89%E5%BF%97)
2. Append '?action=render' to the url, e.g. (http://coppermind.huiji.wiki/wiki/%E9%A3%93%E5%85%89%E5%BF%97?action=render)
3. That's all.
